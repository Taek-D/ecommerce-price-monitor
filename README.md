# 이커머스 가격·재고 실시간 모니터링 자동화

복수 이커머스 플랫폼(무신사, 올리브영, 지마켓, 29CM 등등)에서 상품 가격·재고를 **자동으로 수집하고 변화 시 실시간 알림**을 보내는 Python 기반 자동화 시스템입니다.


## 🎯 주요 기능

- ✅ **멀티플랫폼 모니터링**: 무신사, 올리브영, 지마켓, 29CM 등 이커머스 사이트 동시 감시
- ✅ **실시간 알림**: 가격/품절 변화 감지 시 Discord 즉시 통보
- ✅ **Google Sheets 연동**: 수집한 데이터를 시트에 자동 기록
- ✅ **자동화**: 5분 주기 배치 실행, 24/7 무인 운영
- ✅ **데이터 분석 용이**: 변화 시각 기록으로 시계열 분석 가능


## 📊 성과

| 항목 | 이전 | 이후 | 개선율 |
|------|------|------|--------|
| **수동 작업 시간** | 상품 당 3분 | 1분 내외 | 7~80% ↓ |
| **가격 변화 감지** | 수시간 이상 | 5분 이내 | 96% ↑ |
| **데이터 정확도** | 90% (휴먼에러) | 99%+ (자동화) | +9%p |


## 🛠️ 기술 스택

- **언어**: Python 3.11+
- **웹 자동화**: Playwright (JavaScript 렌더링)
- **데이터 저장**: Google Sheets API
- **알림**: Discord Webhook
- **스케줄링**: APScheduler
- **비동기**: asyncio, httpx


## 🚀 사용 방법
1. 프로그램 실행
2. Google Sheets '상품 URL'열에 상품 URL 입력
3. 5분마다 자동으로 가격/재고 확인
4. 변화 시 Discord 알림 + 시트 자동 갱신

## ⚙️ 환경 설정 (필수)
### Google Service Account 설정
1. Google Cloud Console 접속
2. 새 프로젝트 생성
3. Google Sheets API 활성화
4. 서비스 계정 생성
5. JSON 키 다운로드 → safe/service_account.json 저장

### Discord Webhook 생성
1. Discord 서버 → 설정 → 웹훅
2. 채널별 새 웹훅 생성
3. Webhook URL → .env 파일에 입력

## 📈 데이터 구조
Google Sheets:
| 컬럼 | 내용 |
|------|------|
| **D** | 상품 구매 링크 (입력) |
| **H** | 수집된 가격 (자동 갱신) |
| **J** | 마지막 변화 시간 (자동 기록) |

## 🔧 커스터마이징
코드 상단의 상수들을 수정:
- **MIN_PRICE = 5000**                      # 최소 인식 가격
- **WEB_TIMEOUT = 120000**              # 페이지 로드 타임아웃 (ms)
- **URLS_RELOAD_MINUTES = 30**              # URL 재로드 주기 (분)
