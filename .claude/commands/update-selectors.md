# 셀렉터 업데이트

쇼핑몰 사이트 변경으로 셀렉터가 깨진 경우, 새 셀렉터를 찾아 업데이트합니다.

## 입력

$ARGUMENTS (어댑터 이름 또는 상품 URL)

## 수행 절차

### 1. 대상 어댑터 확인

인자로 어댑터 이름(musinsa, oliveyoung 등)이나 URL을 받아 해당 어댑터를 특정합니다.

### 2. 현재 셀렉터 확인

`musinsa_price_watch.py` 상단의 셀렉터 상수를 읽어 현재 값을 확인합니다.

### 3. 실제 페이지 분석

Playwright로 해당 쇼핑몰의 상품 페이지를 방문하여:
- 가격이 표시되는 DOM 요소의 셀렉터 탐색
- 품절 표시 요소의 셀렉터 탐색
- 기존 셀렉터가 여전히 유효한지 확인

### 4. 셀렉터 업데이트

변경이 필요한 경우:
- 파일 상단의 셀렉터 상수를 업데이트
- 어댑터 클래스 내부에서 사용하는 셀렉터도 함께 확인
- 변경 전후를 사용자에게 보고
